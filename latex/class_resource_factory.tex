\doxysubsection{Detailed Description}
An abstract factory class for creating different types of resources. 

The \doxylink{class_resource_factory}{Resource\+Factory} class provides an interface for creating resources such as \doxylink{class_water}{Water}, \doxylink{class_wood}{Wood}, \doxylink{class_steel}{Steel}, \doxylink{class_coal}{Coal}, and \doxylink{class_concrete}{Concrete}. Derived factory classes should implement the {\ttfamily create\+Resource} method to instantiate specific resource types based on the provided type and capacity.
