\doxysubsection{Detailed Description}
Abstract base class representing the state of a \doxylink{class_building}{Building}. 

The \doxylink{class_building_state}{Building\+State} class defines the interface for handling state-\/specific behavior of a \doxylink{class_building}{Building}. Derived classes must implement the abstract methods to define specific state behaviors and transitions.
