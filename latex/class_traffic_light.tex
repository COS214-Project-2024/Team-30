\doxysubsection{Detailed Description}
Represents a traffic light with a state that can change. 

The \doxylink{class_traffic_light}{Traffic\+Light} class manages the state of a traffic light, which can be red, yellow, or green. It also provides a unique identifier for each traffic light instance.
