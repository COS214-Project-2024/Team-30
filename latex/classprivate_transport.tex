\doxysubsection{Detailed Description}
Represents a private mode of transport that inherits from the \doxylink{class_transport}{Transport} class. 

The \doxylink{classprivate_transport}{private\+Transport} class is a specific type of transport that allows for travel between buildings. It inherits from the \doxylink{class_transport}{Transport} class and implements travel capabilities for private transport options.
