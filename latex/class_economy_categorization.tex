\doxysubsection{Detailed Description}
A strategy class that categorizes the economic state of the city based on \doxylink{class_government}{Government} data. 

This class implements the {\ttfamily \doxylink{class_categorization_strategy}{Categorization\+Strategy}} interface to provide a categorization strategy specifically for the economy of the city.

\item "{}\+Medium Economic Activity"{} if employment rate is between 45\% and 70\%
\item "{}\+High Economic Activity"{} if employment rate is above 70\%
