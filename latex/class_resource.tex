\doxysubsection{Detailed Description}
An abstract base class representing a general resource with a limited capacity. 

The \doxylink{class_resource}{Resource} class provides a common interface for various types of resources such as \doxylink{class_steel}{Steel}, \doxylink{class_water}{Water}, \doxylink{class_coal}{Coal}, \doxylink{class_concrete}{Concrete}, and \doxylink{class_wood}{Wood}. Derived classes must implement methods for getting capacity, using, and refilling the resource.
