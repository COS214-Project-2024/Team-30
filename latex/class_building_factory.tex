\doxysubsection{Detailed Description}
Abstract factory class for creating \doxylink{class_building}{Building} objects. 

The \doxylink{class_building_factory}{Building\+Factory} class provides an interface for creating different types of buildings. Derived factory classes must implement the create\+Building method to instantiate specific building types.
