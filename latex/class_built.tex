\doxysubsection{Detailed Description}
Represents the "{}\+Built"{} state of a \doxylink{class_building}{Building}. 

The \doxylink{class_built}{Built} class implements state-\/specific behavior for buildings that are fully constructed and operational. It provides concrete implementations of the methods defined in the \doxylink{class_building_state}{Building\+State} interface for managing this state.
