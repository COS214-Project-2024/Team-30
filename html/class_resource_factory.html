Detailed Description

An abstract factory class for creating different types of resources.

The ResourceFactory class provides an interface for creating resources such as Water, Wood, Steel, Coal, and Concrete. Derived factory classes should implement the createResource method to instantiate specific resource types based on the provided type and capacity.
